# Workflow: Employee Referral System

The **Employee Referral System** is a platform where current employees can view open positions within the organization and refer candidates.

---

## 📈 Functional Flow
1.  **Job Postings:** Information about current vacancies is managed and displayed on the `Referral.aspx` page.
2.  **Special Incentives:** The system highlights specific referral drives (e.g., "Special Employee Referral Program") with incentive details and validity dates.
3.  **Referral Methods:** 
    *   Employees can view detailed Job Descriptions (JDs) for roles like Business Analysts, .NET Developers, and PHP Developers.
    *   Referrals are handled manually via email to the HR team. The documentation in the app provides the contact details (`anupam@ikf.co.in`) for direct submissions.

---

## 🛠️ Technical Components
*   **Page:** `Referral.aspx`
*   **Styling:** Uses `css/ikf.css` and specialized classes like `.special_employee` for featured job lists.
*   **Media:** Integrated with Milonic JS menu and thumbnail viewers for a consistent site-wide navigation experience.

---

## 🎯 Key Roles Documented
*   **Business Analyst:** Focusing on requirement gathering and stakeholder management.
*   **ASP.NET Software Engineer:** Focusing on N-Tier architectures and SQL optimization.
*   **PHP Developer:** Open-source platform development.
*   **Testing Engineer:** GUI, Functional, and Regression testing.
