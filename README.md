# Timesheet System Documentation

## Introduction
This repository contains the technical and functional documentation for the **Timesheet24** project, an internal HRMS solution designed for employee attendance and leave management.

## Purpose of this Repository
The primary goal of this documentation repo is to provide a clear, structured guide for developers and stakeholders to understand the legacy ASP.NET WebForms architecture. It serves as a central knowledge base to assist in troubleshooting, feature enhancements, and system maintenance.

## Project Overview
**Timesheet24** is a comprehensive Human Resource Management System (HRMS) built on the .NET Framework. It automates critical organizational workflows, including:
- **Attendance Tracking:** Recording daily employee punch-ins and labor hours.
- **Leave Management (LMS):** Processing leave applications, balance accruals, and multi-level approvals.
- **Reporting:** Generating MIS and performance analytics via Crystal Reports integration.
- **Employee Portal:** Providing self-service tools for profile updates and internal communications.
